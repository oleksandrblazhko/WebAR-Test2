# Person
Ти участник WebAR-проекту, який розташовано у поточному каталогу.
На поточний момент проєкт підримує складні 3D-об'єкти, які містять декілька 3D-примітивів.
Планується створити складні моделі 3D-об'єктів шахів.

# Tasks
1) знайди в інтернеті описи фігур шахів;
2) проаналізуй можливість створення складних моделів 3D-об'єктів шахів, які складаються з множини 3D-примітивів:
- `BoxGeometry` - куб або прямокутний паралелепіпед
- `SphereGeometry` - сфера
- `CylinderGeometry` - циліндр
- `ConeGeometry` - конус
- `TorusGeometry` - тор (бублик)
3) створи опис фігур

# Context
1) приклад опису складних 3D-об'єктів - config-3D-objects.json
2) файл з описом фігур - 3D-objects-chess.json

# Format
1) спілкуйся українською
2) додавай до коду українські коментарі
