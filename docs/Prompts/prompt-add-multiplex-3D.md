# Person
Ти участник WebAR-проекту, який розташовано у поточному каталогу.
На поточний момент проєкт підримує візуалізацію 3D-примітивів з бібліотеки threex.
Але треба створмити можливість підтримувати складні 3D-об'єкти, які містять декілька 3D-примітивів.

# Tasks
1) проаналізуй можливий приклад опису складних 3D-об'єктів;
2) додай можливість описувати у конфігураційному файлі складні 3D-об'єкти з використанням "contentType": "multiplex-3D";
3) додай до конфігураційного файлу опис barcode-маркеру з "target": 2, який буде описувати "contentType": "multiplex-3D" 
для складного об'єкту "object_id": 1 
4) додай до основного файлу обробку "contentType": "multiplex-3D"

# Context
1) приклад опису складних 3D-об'єктів - config-3D-objects.json
2) конфігураційний файл - config.json
3) під час візуалізації складного 3D-об'єкта розглядай алгоритм, який:
- спочатку візуалізує 3D-примітив зі значенням "part_id": 1, який розташовується на площині маркера
- 3D-примітив зі значенням "part_id": 2 розташовується на площині попереднього 3D-примітиву (зверху примітиву)
- наступний 3D-примітив розміщується зверху попереднього і так далі. 
4) основний файл - index-1.html

# Format
1) спілкуйся українською
2) додавай до коду українські коментарі
