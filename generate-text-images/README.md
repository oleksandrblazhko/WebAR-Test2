# Генерація зображень + Speech Synthesizer

Інтегрована програма для створення JPG-зображень з текстом та їх озвучення.

## Використання

Просто відкрийте файл `index.html` у браузері (Chrome, Firefox, Edge).

**Ніяких додаткових встановлень не потрібно!**

## Можливості

### 🎨 Генератор зображень:
- ✅ Генерація зображень з текстом
- ✅ Налаштування розміру, кольору, шрифту
- ✅ Попередній перегляд всіх зображень
- ✅ Масове завантаження у ZIP-архіві
- ✅ Індивідуальне завантаження кожного зображення

### 🔊 Speech Synthesizer:
- ✅ Озвучення тексту (Text-to-Speech)
- ✅ Вибір голосу (авто-вибір іспанської мови)
- ✅ Налаштування швидкості та тону
- ✅ Озвучення всіх слів послідовно

## Як використовувати

1. Відкрийте `index.html` у браузері
2. Введіть слова у текстове поле (кожне з нового рядка)
3. Налаштуйте параметри зображень за бажанням
4. Натисніть **"🚀 Створити зображення"**
5. Перегляньте результат у попередньому перегляді
6. Для озвучення:
   - Введіть текст у поле Speech Synthesizer та натисніть 🔊
   - Або натисніть **"🔊 Озвучити всі"** для послідовного озвучення
   - Або натисніть 🔊 на картці зображення
7. Натисніть **"📥 Завантажити ZIP"** для масового завантаження

## Гарячі клавіші

- **Enter** у полі вводу тексту → озвучити текст

## Вихідні файли

Зображення зберігаються у ZIP-архіві з іменами файлів, що відповідають словам (пробіли замінюються на `_`).
